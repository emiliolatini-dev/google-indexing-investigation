## Purpose of this chapter

Describe how the investigation is conducted so that it can be reproduced: the
epistemic categories (FACT / OBSERVATION / HYPOTHESIS / QUESTION), the evidence
collection and immutability rules, the identifier system, and the criteria used
to accept or reject a hypothesis. This chapter documents *process*, not findings.
