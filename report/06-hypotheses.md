## Purpose of this chapter

Present possible explanations for the observed behavior as HYPOTHESIS (`H-`)
entries — each explicitly provisional, each listing the evidence that would
support or falsify it, and each with a test plan. No hypothesis is stated as a
fact. This chapter mirrors and narrates the entries tracked in
[`registry/hypotheses.md`](../registry/hypotheses.md).
